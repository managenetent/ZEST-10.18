3. NEW OPS (3 operations built from scratch):

   a) ops/screener.c (~210 lines)
      - Scanned all corp_*/state.txt files to collect corporation data
      - Calculated fundamental value using real analysis_loop.c formula:
        * book_value_per_share = book_value / shares_outstanding
        * market_cap_multiplier = 1.0 + (log10(market_cap) - 3) * 0.05
        * leverage_factor = 1.0 - (debt_to_equity - 1.0) * 0.1 [if debt > 1.0]
        * bias_factor = 0.5 + (risk_bias / 100.0) * 1.0
        * fundamental_value = book_value_per_share * multipliers
      - Calculated upside_pct = ((fundamental_value - price) / price) * 100
      - Sorted by upside percentage (best opportunities first)
      - Displayed: Ticker | Price | Fundamental | Upside% | MktCap | P/E | Leverage
      - Showed top 50 stocks
      - Saved output to pieces/wsr_menu/screener_report.txt
      - Output one-line message: "Screener report saved - view via `cat ...`"

   b) ops/player_list_portfolio.c (~170 lines)
      - Read transactions.txt (TICKER|ACTION|SHARES|PRICE|DATE format)
      - Calculated current holdings per ticker (sum of buys/sells/shorts/covers)
      - Calculated average cost basis (sum of buy costs / current shares)
      - Read current stock prices from corp_*/state.txt
      - Outputted formatted table: Ticker | Shares | Avg Cost | Current | Gain/Loss $ | Gain/Loss %
      - Showed total portfolio value and aggregate gain/loss
      - Saved output to pieces/wsr_menu/portfolio_report.txt
      - Output one-line message: "Portfolio report saved - view via `cat ...`"

   c) ops/wsr_search_input.c (~100 lines)
      - Read/wrote search criteria from wsr_search_menu/state.txt
      - Managed fields: high_roe, low_pe, price_pct_book, div_yield, min/max_market_cap, etc.
      - Handled criterion 15 (Clear Criteria) to reset all filters to defaults
      - Stub for interactive input (full stdin handling would require architectural change)

   d) ops/display_search_results.c (~130 lines)
      - Read search criteria from wsr_search_menu/state.txt
      - Scanned all corp_*/state.txt files in projects/wsr-pal/pieces/
      - Applied filters: market cap range, price as % of book value, publicly_traded flag, has_bonds flag
      - Displayed matching corporations in table format
      - Showed: Ticker | Price | BookValue | P/E | ROE | MktCap | Div
