FONDU COMMAND REFERENCE (VERIFIED)
================================================================================

Basic Usage:
  ./pieces/system/fondu/fondu.+x <command> [project_name]

Commands:
  --install <project>    : Compile, deploy to installed/, register ops
  --uninstall <app>      : Remove from installed/, unregister ops
  --archive <project>    : Move to trunk/ (source only, not compiled)
  --restore <project>    : Move from trunk/ back to projects/, recompile
  --list                 : Show all projects and their states
  --list-ops             : Show all available ops (from ops catalog)
  --help, -h             : Show help message

What Each Command Does:

┌──────────────┬─────────────────────┬──────────────────────┬──────────┬───────────┐
│ Command      │ Source              │ Destination          │ Compile? │ Editable? │
├──────────────┼─────────────────────┼──────────────────────┼──────────┼───────────┤
│ --install    │ projects/           │ installed/           │ Yes      │ No        │
│ --uninstall  │ installed/          │ (deleted)            │ N/A      │ N/A       │
│ --archive    │ projects/           │ projects/trunk/      │ No       │ Yes       │
│ --restore    │ projects/trunk/     │ projects/            │ Yes      │ Yes       │
│ --list       │ N/A                 │ stdout               │ No       │ N/A       │
│ --list-ops   │ ops_catalog.txt     │ stdout               │ No       │ N/A       │
└──────────────┴─────────────────────┴──────────────────────┴──────────┴───────────┘
