================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Command not found"
Solution: Make sure you're in the project root directory

Problem: "Permission denied"
Solution: Run: chmod +x projects/p2p-net/ops/+x/*.+x

Problem: "Cannot create config file"
Solution: Check that pieces/network/subnets/ directory exists

Problem: "No peers found"
Solution: Make sure both terminals joined the same subnet

Problem: "Message not delivered"
Solution: Verify peer is connected with list_peers first
