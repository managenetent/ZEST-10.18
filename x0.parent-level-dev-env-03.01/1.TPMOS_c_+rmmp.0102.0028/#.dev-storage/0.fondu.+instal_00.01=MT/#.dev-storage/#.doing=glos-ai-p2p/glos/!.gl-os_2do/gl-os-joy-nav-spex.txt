```
╔═══════════════════════════════════════════════╗
║  Win:1  Act:1  M(0,0) S=0                    ║
╠═══════════════════════════════════════════════╣
║  MENU (Joystick UP/DOWN to navigate)         ║
║  ║  [>] 1. [Desktop Apps]                    ║  ← Extra ║, always [>]
║  ║  [>] 2. [Terminal Apps]                   ║  ← Extra ║, always [>]
╚═══════════════════════════════════════════════╝
```

**Problems:**
1. ❌ BOTH items show `[>]` (should be only ONE)
2. ❌ Extra `║` at start of each menu line
3. ❌ Selection doesn't change when joystick is used
4. ❌ Frame doesn't update when joystick moves selection

## REQUIRED FIXES

### 1. Fix Menu Format

**Remove extra `║` from menu generation:**

```c
/* WRONG - includes extra ║ */
snprintf(line, sizeof(line), "║  %s %d. [%s]                            ║\n", 
         indicator, i + 1, menu_items[i]);

/* RIGHT - no extra ║ */
snprintf(line, sizeof(line), "  %s %d. [%s]\n", 
         indicator, i + 1, menu_items[i]);
```

### 2. Fix Selection State

**Joystick must update `selected_index`:**

```c
void joystick(unsigned int buttonMask, int x, int y, int z) {
    if (active_window_id != -1) {
        for (int i = 0; i < window_count; i++) {
            if (windows[i].id == active_window_id && 
                strcmp(windows[i].type, "terminal") == 0) {
                
                /* Y axis - move selection */
                if (y < -500 && last_joy_y >= -500) {
                    /* Joystick UP - move selection up */
                    windows[i].selected_index--;
                    if (windows[i].selected_index < 0) 
                        windows[i].selected_index = 0;
                    
                    /* WRITE NEW FRAME */
                    write_gl_os_state();
                    parse_and_write_frame();
                    
                } else if (y > 500 && last_joy_y <= 500) {
                    /* Joystick DOWN - move selection down */
                    windows[i].selected_index++;
                    if (windows[i].selected_index >= 2) 
                        windows[i].selected_index = 0;  /* Wrap around */
                    
                    /* WRITE NEW FRAME */
                    write_gl_os_state();
                    parse_and_write_frame();
                }
                last_joy_y = y;
            }
        }
    }
    glutPostRedisplay();
}
```

### 3. Fix Frame Writing

**State file must have CORRECT selected_index:**

```c
void write_gl_os_state(void) {
    /* ... */
    
    /* Get selected_index from terminal window */
    int selected_idx = 0;
    for (int i = 0; i < window_count; i++) {
        if (windows[i].id == active_window_id && 
            strcmp(windows[i].type, "terminal") == 0) {
            selected_idx = windows[i].selected_index;
            break;
        }
    }
    
    /* Build menu with CORRECT selection */
    for (int i = 0; i < 2; i++) {
        const char *indicator = (i == selected_idx) ? "[>]" : "[ ]";
        /* ... */
    }
}
```
