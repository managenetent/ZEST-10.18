================================================================================

Priority: CRITICAL (foundation for everything else)

[ ] 6.1: Network Infrastructure (7 ops)
    ───────────────────────────────────
    [ ] p2p::configure_subnet - Configure network mode (broadcast/ring/hybrid)
    [ ] p2p::broadcast_join - Join network via UDP broadcast
    [ ] p2p::broadcast_leave - Leave network (graceful)
    [ ] p2p::connect_peer - TCP connection to peer
    [ ] p2p::disconnect_peer - Disconnect from peer
    [ ] p2p::list_peers - List connected peers
    [ ] p2p::ping_peer - Keepalive (auto every 30s)
    
    KPIs:
    ✓ Subnet config loaded from subnet-config.txt
    ✓ Port discovery works (8000-8010)
    ✓ Broadcast JOIN/LEAVE works (UDP 255.255.255.255:8000)
    ✓ TCP connections established (handshake: Hello→Ack)
    ✓ Peers list updated (pieces/network/peers.txt)
    ✓ Continuous mode: background listener thread
    ✓ On-demand mode: connect/disconnect per op
    ✓ Default hosts checked on startup
    
    Test Commands:
    OP p2p::configure_subnet "general" "hybrid" "continuous" 8000
    OP p2p::broadcast_join
    OP p2p::list_peers "all"
