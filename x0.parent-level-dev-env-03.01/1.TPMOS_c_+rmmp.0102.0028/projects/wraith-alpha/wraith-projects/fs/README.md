# Wraith FS

Internal Wraith filesystem project scaffold.

Purpose:
- establish `fs` as a real Wraith-hosted project
- keep one project identity for filesystem work
- prove ASCII first using the `agy-text-editor` browser pattern
- later add a CHTPMGL surface over the same fs ops/state

Current status:
- scaffold only
- no dedicated manager or project-owned ops yet
- no separate `ascii-fs` / `gl-fs` split
