# TPMOS Testing Experience Log - Agent Learnings

This log captures key learnings and successful interaction patterns encountered by the agent during testing of the TPMOS CLI, specifically for navigating the main menu and interacting with system features via key injection.

**Date of Entry:** 2026-04-21

**Summary of Experience:**
Successfully navigated from the CHTPM CLI main menu to the "Project Loader" screen by correctly injecting key presses. This involved overcoming initial challenges with the key injection format and the `date` command.

**Key Learnings and Procedures:**

1.  **System Startup:**
    *   The CHTPM orchestrator is launched using `./button.sh run`.
    *   This presents a text-based main menu.

2.  **Frame Viewing:**
    *   Frame updates are logged in `pieces/debug/frames/session_frame_history.txt`.
    *   `current_frame.txt` may also show the latest frame.

3.  **Key Injection for Navigation:**
    *   **Required Format:** The precise format for injecting key presses is `[YYYY-MM-DD HH:MM:SS] KEY_PRESSED: <code>`.
    *   **Timestamp Issues:** Dynamic generation of timestamps using the `date` command in the shell environment can be problematic due to parsing errors or command availability. Using a fixed, valid timestamp is a reliable workaround.
    *   **Successful Key Codes:**
        *   'Enter' key: `KEY_PRESSED: 13` (decimal code). This successfully navigated to the Project Loader.
        *   '1' key (for menu option 1): Initial attempts to inject '1' did not result in frame updates. Further investigation may be needed on the specific code or context required for this key.
    *   **Example Injection (for Enter):**
        ```bash
        echo "[2026-04-21 08:08:15] KEY_PRESSED: 13" >> pieces/keyboard/history.txt
        ```

4.  **Interaction Flow:**
    *   Launched system: `./button.sh run`
    *   Observed main menu.
    *   Attempted to inject '1' (failed to update frame).
    *   Successfully injected 'Enter' (code 13) with correct format, which resulted in frame update (user confirmed this).
    *   Current state: Navigated to Project Loader.
