# Network, LSR, and Multiplayer
Date: 2026-06-24

## Purpose

This lane is the shared network and social economy layer:

- `p2p-net`
- login / identity
- mining / trading / chat
- multiplayer state
- LSR / civilization simulation

## Key Concern

This should not be built too early if the core project contract is still changing.
Networking magnifies mistakes.

## What LSR Suggests

LSR already points toward:

- curriculum / learning
- agents
- map and z-layer expansion
- recursive knowledge growth

That makes it a natural long-term home for civilization-scale simulation logic.

## Short-Term Goal

Do not try to ship the full network civilization at once.
Instead, prove a small real network loop that can survive contact with the existing state model.

## Success Signatures

- identity is stable
- state can be shared or synchronized
- chat and trading use the same core contracts
- multiplayer does not require rewriting the engine later
